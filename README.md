# Moon Image Simulation

​	For CCD image simulation






## How to use?

1.  **Set the parameters(or dataset) below:**
   - Orbit(Kepler elements).
   - Camera(CCD camera). You can add as many cameras as you want.
   - Moon(DOM  and DEM). 

​	

2. **Then Click *generate*.**  



## Dependencies

- GDAL
